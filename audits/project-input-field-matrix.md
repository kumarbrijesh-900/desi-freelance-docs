# Project Input Field Matrix

Scope note:
- This matrix covers user-editable data inputs and file inputs discovered in the current repo.
- It includes the canonical invoice flow (`components/invoice/*`) and the frozen legacy `/create` wizard (`components/flow/*`).
- Action-only buttons such as `Save draft`, `Preview & download`, `Magic Fill`, `Continue with Google`, section navigation, and placeholder voice triggers are excluded because they do not directly bind to a persistent input property.
- Where a UI control writes to one property and also triggers sync/derived side effects, the main destination is listed first and the side effect is called out explicitly.
